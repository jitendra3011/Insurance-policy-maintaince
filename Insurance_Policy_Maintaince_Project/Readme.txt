How to run Insurance Policy Maintaince Project
Steps:
1. Download the  zip file
2. Extract the file and copy ims folder
3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name ipmdb
6. Import ipmdb.sql file(given inside the zip package in sql file folder)
7.Run the script http://localhost/Insurance_Policy_Maintaince   (frontend)
8. For admin panel http://localhost/Insurance_Policy_Maintaince/admin  (admin panel)
